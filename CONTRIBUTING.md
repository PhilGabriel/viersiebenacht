# Beitragende-Richtlinien

Vielen Dank, dass du zu diesem Projekt beitragen möchtest! Hier sind einige Richtlinien, die dir helfen sollen:

## Wie du beitragen kannst

1. **Fehler melden**: Nutze das [Issue-Tracker](https://github.com/PhilGabriel/viersiebenacht/issues) für das Melden von Fehlern.
2. **Feature-Anfragen**: Eröffne ein Issue mit einer klaren Beschreibung deiner Idee.
3. **Pull Requests**: 
   - Stelle sicher, dass dein Code gut dokumentiert und getestet ist.
   - Erkläre, warum du Änderungen vornimmst.
   
## Code-Style

- Achte auf sauberen und gut formatierten Code.
- Folge den bestehenden Stilrichtlinien im Projekt.

## Tests

Füge immer Tests für neue Funktionen oder Fehlerbehebungen hinzu.

Vielen Dank für deine Mithilfe!
