### Project Description
This project offers a simple yet effective tool to support the **4-7-8 breathing technique**. This breathing exercise helps reduce stress and promotes relaxation. The tool provides clear visual cues for when to inhale, exhale, and hold your breath based on the 4-7-8 rhythm.

### Features
- **Easy to use**: Click "Start" and follow the instructions to begin the breathing exercise.
- **Visualized Breathing Phases**: Color changes and a countdown guide the user through the exercise.
- **Explanation of the 4-7-8 Technique**: A separate page provides a detailed explanation of the method.
- **Responsive Design**: Optimized for various devices and screen sizes.
- **Social Media Sharing**: The page includes social media metadata and a suitable image for sharing.

### Technologies Used
- **[Bootstrap](https://getbootstrap.com/)** for responsive design and layout.
- **[jQuery](https://jquery.com/)** for DOM manipulation and event handling.
- **CSS3** for custom styles.
- **HTML5** for structure and semantics.

### Pages
1. **Main Page**: The home page with the breathing exercise, countdown, and instructions.
2. **Background Page**: An explanation of the 4-7-8 breathing technique and its benefits.

### Installation
To use the project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/PhilGabriel/viersiebenacht.git
    ```
2. Navigate to the project directory:
    ```bash
    cd viersiebenacht
    ```
3. Open the `index.html` file in your browser.

### Folder Structure
- `/css/`: Contains the CSS file for the design.
- `/js/`: Contains the JavaScript file for the breathing exercise.
- `/img/`: Contains the social media image.
- `index.html`: The main page with the 4-7-8 breathing exercise.
- `hintergrund.html`: A page explaining the breathing technique.
